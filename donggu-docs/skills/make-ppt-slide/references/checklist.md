# 배포 전 체크리스트

덱을 전달하기 전에 통과시킨다.

## 필수 검사

- 모든 슬라이드에 등록된 `data-layout`.
- 본문 페이지는 `S01`~`S22`만.
- 지도·로케이션 페이지는 `S08 + Tightened Map Component`.
- 이미지 분할·evidence wall·장식 기하용 커스텀 본문 레이아웃 발명 X.
- 모든 로컬 이미지에 `data-image-slot`.
- `S22`는 `data-image-slot="s22-hero-21x9"`.
- SVG 안에 가시 `<text>` 없음.
- 검증기 통과:

```bash
node scripts/validate-deck.mjs path/to/index.html
```

## 언어 검사

- 덱이 한국어 모드 또는 영어 모드 중 하나로 일관됨.
- 영어 덱: `<html lang="en" data-language="en">`.
- 한국어 덱: `<html lang="ko" data-language="ko">`.
- 한국어 덱은 한국어 폰트 스택 사용: `SUIT`, `Pretendard`, `Noto Sans KR`, `Noto Sans`.
- 다이어그램 라벨과 이미지 내 텍스트가 선택된 언어와 일치.

## 시각 시스템

- 액센트 컬러 1개만.
- 그라디언트·그림자·글래스·둥근 카드·네온·장식 보더 없음.
- 큰 타이틀은 라이트 웨이트(`200` 또는 `300`).
- 큰 폰트는 `font-size:min(Xvw,Yvh)`, `vh` 헤드룸 확보.
- 본문 슬라이드 타이틀이 좌측·상단 콘텐츠 축에 정렬됨 (statement·split 제외).
- kicker·meta 텍스트가 타이틀 위에 위치.
- `.canvas-card` 가로 패딩이 중첩되지 않음 (`padding:... 5vw ...` 중복 금지).
- 콘텐츠가 하단 네비게이션 안전 영역을 침범하지 않음.

## 이미지 검사

- 단일 큰 비주얼은 `S22`.
- 다중 이미지 페이지는 `S15`·`S16` 적응.
- 한 그룹 안에서 이미지 비율·시각 스케일 통일.
- `S22` 사진은 `object-position:center 35%` 또는 `center center`. `top center` 금지.
- UI·인포그래픽 이미지는 원본 보존이 필요할 때만 `fit-contain`.
- 재생성 슬롯 이미지는 비율 프레임을 가득 채움 (예: `.frame-img.r-21x9`).
- 생성된 이미지에 슬라이드 타이틀·푸터·페이지 번호·코너 마크·로고·서명·보더 없음.

## 브라우저 검사 (사용자 직접 수행)

- 애니메이션 안착 후 모든 페이지 점검.
- 화살표·휠·터치(해당 시)·`ESC` 인덱스·`B` 정적 모드 네비게이션.
- `ESC` 인덱스 썸네일이 애니메이션된 콘텐츠를 보여줌.
- 캡션·타임라인 라벨·각주·KPI 블록이 네비게이션 닷과 충돌하지 않음.

## donggu 추가 점검 (강의 컨텍스트)

- 한국어 폰트가 실제로 적용되었는지 (PC·맥에서 SUIT/Pretendard 로딩 확인).
- 강사 사진·강의장 사진이 들어간 경우 라이선스·초상권 문제 없음.
- 강의 슬라이드라면 청중 시야 거리 기준 폰트 크기 점검 (큰 강의실 = 더 큰 본문 폰트 권장).
- 발표 모드(`B`)에서 청중에게 보이지 않을 ESC 인덱스가 노출되지 않는지 사전 리허설.
