# kr-card-principles — 한국어 / 한국 카드뉴스 보정

DESIGN.md가 글로벌(영문) 브랜드라도 한국어 카드뉴스로 렌더할 때 이 규칙을 겹쳐 적용한다. 출처: 한국 카드뉴스 제작 정석 + 한글 웹폰트 실무.

## 폰트 대체 (DESIGN.md 폰트가 한글 미지원일 때)

DESIGN.md의 **역할 분리(display / body)는 유지**하되, 한글 글리프는 아래로 매핑한다. 라틴 폰트(Inter, BMW Type 등)는 한글이 없어 깨지므로 fallback이 필수.

| DESIGN 역할 | 한글 대체 (CDN) |
|---|---|
| Display heavy / 고딕 임팩트 | **Pretendard** 700~900 (`cdn.jsdelivr.net/gh/orioncactus/pretendard`) · 더 강한 임팩트는 G마켓 산스 Bold |
| Body / 라이트 | **Pretendard** 300~500 |
| Display 명조/세리프 | **Noto Serif KR** |
| 라틴 라벨·숫자 (UPPERCASE 라벨, 페이지번호) | DESIGN.md 원본 라틴 폰트 그대로 (Inter 등) — 한글 안 섞이는 곳 |

원칙: **한글에 UPPERCASE·과도한 letter-spacing 금지** (자모가 깨져 보임). UPPERCASE·tracking은 라틴 라벨("VIEW MORE", "VOL. 01")에만.

## 크기 위계 (1080×1350 기준, 폰에서 읽힘)

웹 px를 그대로 쓰면 폰에서 작다. 카드 스케일로:

| 역할 | 권장 px | 비고 |
|---|---|---|
| 표지 메인 타이틀 | 88~130 | 2~4줄, 한 줄에 너무 많은 글자 금지 |
| 본문 카드 제목 | 56~80 | |
| 본문 텍스트 | 30~40 | **28 미만 금지** (폰 가독성 하한) |
| 라벨/페이지번호/메타 | 22~30 | |

## 색·구성 정석 (DESIGN.md가 비어있을 때의 기본값)

- 색 **3~4개로 제한** (배경 / 잉크 / 포인트 1~2). COOLORS 팔레트 감각.
- 표지가 제일 중요 — 주제를 한눈에. 주제 연관 비주얼 요소 1개.
- 본문/제목은 **고딕**이 기본 (깔끔·전달력). 포인트만 다른 폰트.
- 이모지/픽토그램은 키워드 강조에만 (남발 금지). 단, **DESIGN.md가 장식 금지면 쓰지 않는다.**
- 브랜드 요소(로고·핸들·시그니처 색)를 매 카드 푸터에 일관되게.

## 한글 조판 디테일
- 모든 텍스트 컨테이너 `word-break: keep-all` (단어 단위 줄바꿈 — 어절이 중간에 안 끊김).
- 줄높이 본문 1.5~1.65, 제목 1.2~1.35.
- 문장부호 끝 정렬 주의 — 따옴표·온점이 줄 끝에 혼자 남지 않게 `<br>` 수동 조정.
